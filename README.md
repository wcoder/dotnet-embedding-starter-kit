# WIP: .NET Embedding Starter Kit
[.NET Embedding](https://docs.microsoft.com/en-us/xamarin/tools/dotnet-embedding/) starter kit for mobile development.


## Build E4K from sources
Build for Android:
```
build-e4k.sh android
```
Build for iOS:
```
build-e4k.sh ios
```
Build for all:
```
build-e4k.sh
```
